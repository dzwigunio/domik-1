# Parametry domu pasywnego (Darmstadt)

| Cecha | Wartość / Wymóg |
| ----- | ----- |
| **Zapotrzebowanie na energię cieplną** | max. 15 kWh/m² w skali roku (co odpowiada \< 1.5m³ gazu lub 5 kWh prądu z pompą ciepła) |
| **Izolacyjność termiczna (Współczynnik U)** | docelowo 0,1 W/(m²K), maksymalnie 0,15 W/(m²K) dla całej bryły budynku |
| **Szczelność powietrzna budynku (n50)** | Wynik testu ciśnieniowego poniżej 0,6 wymiany kubatury na godzinę (\< 0,6 h⁻¹) przy różnicy ciśnień 50 Pa |
| **Sprawność odzysku ciepła (wentylacja)** | Wentylacja mechaniczna z odzyskiem ciepła (rekuperacją) o sprawności powyżej 80% |
| **Zużycie energii przez centralę wentylacyjną** | poniżej 0,45 W/m³ |
| **Współczynnik kształtu bryły (A/V)** | od 0,75 do 1,2 (im mniejszy, tym bardziej zwarta bryła) |
| **Orientacja głównych przeszkleń** | południowa (w celu pasywnego pozyskania energii słonecznej) |
| **Wstępne podgrzewanie powietrza (np. przez GWC)** | do temperatury powyżej 5°C w sezonie zimowym |
| **Przygotowanie ciepłej wody użytkowej** | wykorzystanie odnawialnych źródeł energii (np. kolektory słoneczne, pompy ciepła) |
| **Brak mostków termicznych** | Konstrukcja zaprojektowana tak, aby zminimalizować mostki cieplne, co zapobiega stratom ciepła i kondensacji wilgoci  |
|  |  |
